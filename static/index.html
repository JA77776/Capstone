Express API

Routes information

Routes HTTP methods Description
/ GET Render an index.html
Other routes are protected by middleware. You must login first.

User routes

Routes HTTP methods Description
/ GET To retrieve the data
/:id GET To retrieve specific data
/register POST To update the value
/:id/alterUser PATCH To update the data
/:id/deleteUser DELETE To delete the data

Products Routes

Routes HTTP methods Description
/ GET To retrieve the products
/:id/getProduct GET To get a specific product
/addProduct POST To add the product/s
/:id/alterProduct PATCH To update the data of the product/s
/:id/deleteProduct DELETE To delete the product/s
